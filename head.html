<HEAD>

раздел заголовка HTML-документа
Описание ›

Элемент <head> представляет раздел заголовка HTML-документа и включает в себе теги, содержащие техническую информацию о документе.

В заголовке располагается следующая информация:

  • Название документа;
  • Ключевые слова и описание страницы для поисковых машин;
  • Инструкции для браузера по отображению страницы;
  • Подключение внешних файлов таблицей стиля и скриптов;
  • Описание внутренних таблиц стилей и скриптов.

Эта информация не отображается на экране, но она очень важна для браузера.

Элемент-контейнер <head> содержит информацию о документе организованную в виде коллекции метаданных, которые используются для хранения информации предназначенной для браузеров и поисковых систем. Например, механизмы поисковых систем могут обращаются к метатегам для получения описания сайта, ключевых слов и других данных влияющих на результаты поисковых запросов. Тег <head> располагается между тегами <html>и <body>

Каждый элемент <head> должен содержать элемент <title>, который задает заголовок окна веб-страницы. Внутри контейнера <head> допускается размещать следующие элементы:

  • base
  • basefont
  • bgsound
  • link
  • meta
  • script
  • style
  • title
  • noscript

(Теги BASEFONT и BGSOUND поддерживаются не всеми браузерами, а также осуждаются или не поддерживаются в спецификациях HTML 4.01, HTML 5, XHTML 1.0 и XHTML 1.1.)
